这是一个收集课表，统计公共空暇时间的项目
项目功能虽小但五脏六腑居在
技术栈：
- 前端：（前端打包命令：vite build）
1. vue3.0(组合式API)
2. vite
3. axios
4. vue-router
5. pinia
- 后端：
1. SpringBoot2.7 + springmvc
2. SpringSecurity
3. mysql + redis
4. mybatis-plus
5. jackson
6. swagger
